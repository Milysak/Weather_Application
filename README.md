# Weather_Application
Projekt pogodynka napisany w Kotlinie, korzystający z API - openWeatherApp.

## Technologie

- Android (Kotlin),
- Okhttp - wysyłanie zapytań o pogodę do openWeatherApp,
- Picasso - pobieranie ikonek pogody.
